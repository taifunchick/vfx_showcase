# VFX Showcase Project
**Unity 2022.3.39f1 LTS**

---

## Как открыть проект

1. Установите Unity Hub и Unity 2022.3.39f1 LTS.
2. Нажмите **Open** в Unity Hub.
3. Выберите папку с проектом.
4. Откройте сцену: `Assets/Scenes/Main.unity`.
5. Нажмите **Play**.

---

## Управление

- Нажмите на кнопки **Particle**, **Flipbook**, **Mesh**, **Shader** для выбора эффекта.
- Камера плавно перемещается к выбранному эффекту.
- Слайдер **"Настройки"** изменяет размер текущего объекта.
- Название активного эффекта отображается по центру экрана.

---

## Как заменить объект-заглушку на реальный VFX

1. В иерархии (**Hierarchy**) найдите папку **Zones**.
2. Выберите нужную зону: `Zone_Particle`, `Zone_Flipbook`, `Zone_Mesh`, `Zone_Shader`.
3. Удалите дочерний объект **VFX_Stub**.
4. Добавьте свой VFX-префаб, перетащив его в зону.
5. При необходимости обновите значения в скрипте `VFXSelector`.

**Важно:** новый префаб должен быть дочерним объектом зоны.

---

## Технические детали

- Версия Unity: 2022.3.39f1 LTS.
- Render Pipeline: Built-In.
- Все эффекты легко заменяются через префабы.
- Слайдер сохраняет значение для каждого эффекта отдельно.
- Камера перемещается плавно, без телепортации.
